## E-Commerce Sales Analysis using Pandas

## Project Overview

This project analyzes an e-commerce sales dataset using Python and Pandas. The goal is to practice data analysis techniques and answer business-related questions using real-world data.

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Analysis Performed

* Displayed the top and bottom records of the dataset.
* Checked the number of rows and columns.
* Identified the data types of each column.
* Found the highest, lowest, and average purchase prices.
* Counted customers based on language and job titles.
* Filtered records using different conditions.
* Analyzed email domains and credit card expiration dates.
* Retrieved addresses associated with high purchase amounts.

## Key Learnings

Through this project, I gained hands-on experience with data inspection, filtering, aggregation, string operations, sorting, and extracting insights using Pandas.

## Conclusion

This project helped me strengthen my understanding of Pandas and fundamental data analysis concepts using an e-commerce dataset.
