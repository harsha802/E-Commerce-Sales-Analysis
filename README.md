# E-Commerce Sales Analysis using Pandas

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an e-commerce sales dataset using Python and Pandas. The objective is to analyze customer purchase behavior, identify patterns in transactions, and answer business-related questions using real-world data.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Analysis Performed

* Displayed the top and bottom records of the dataset.
* Determined the number of rows and columns.
* Examined data types and dataset information.
* Found the highest, lowest, and average purchase prices.
* Analyzed customer purchases based on language preferences and job titles.
* Filtered records using multiple conditions.
* Identified the most popular email providers.
* Analyzed credit card expiration patterns.
* Retrieved addresses associated with high purchase amounts.
* Compared customer transactions across different time periods.

## Visualizations

* Purchase Price Distribution Histogram
* Language Distribution Bar Chart
* Top 10 Purchase Prices Visualization
* Top 5 Email Providers Bar Chart

## Key Findings

* Purchase prices varied significantly among customers.
* English was the most commonly used language in the dataset.
* A small number of email providers were used by the majority of customers.
* Customer purchasing patterns differed between AM and PM periods.
* Filtering and aggregation techniques helped uncover meaningful business insights.

## Conclusion

This project strengthened my understanding of Pandas and fundamental data analysis concepts, including filtering, aggregation, string operations, visualization, and extracting insights from real-world e-commerce data.
